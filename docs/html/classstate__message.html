Detailed Description

The class state_message wraps around state_message_t to provide it with accessor methods like iterators that can be used in for loops, etc.

Member Function Documentation

iterator state_message::end ( )

Return pointer to last element.

Note that this iterator is "equality comparable". Only random access iterators are relationally comparable. That's why you always use a for-loop like this:

for (iter = sm.begin(); iter != sm.end(); iter++) { // something with *iter }
